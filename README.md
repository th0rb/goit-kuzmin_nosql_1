3. Що таке $out і чим він відрізняється від $merge? Коли використовувати кожен?

Обидва оператори використовуються наприкінці Aggregation Pipeline для збереження оброблених даних у певну колекцію. Головна різниця — у тому, як вони поводяться з наявними даними.
Оператор $out перезаписує колекцію. Він бере результат роботи пайплайну і повністю замінює ним цільову колекцію (або створює нову, якщо її не існує)
Оператор $merge діє більш гнучко. Замість того щоб знищувати колекцію, він інтегрує (merge) результати агрегації у цільову колекцію

## Частина 2 — Запити до даних

1. Для чого використовується інструкція $unwind?

Інструкція $unwind використовується в Aggregation Pipeline MongoDB для розгортання масивів. Вона перетворює один документ із масивом на кілька окремих документів (по одному для кожного елемента масиву)

2. Чим $stdDevPop відрізняється від $stdDevSamp?

Оператори $stdDevPop та $stdDevSamp у MongoDB використовуються для обчислення стандартного відхилення, але застосовуються для різних типів даних.

$stdDevPop (генеральна сукупність): Використовується, коли дані представляють весь набір (всю сукупність), і вам не потрібно робити висновки про більшу групу. 

$stdDevSamp (вибірка): Використовується, коли ваші дані є лише частиною (вибіркою) більшого набору, і ви хочете оцінити стандартне відхилення для всієї популяції.  Це найчастіший вибір в аналітиці.
